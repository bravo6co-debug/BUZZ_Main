## 🏢 서비스 개요

### 타겟 사용자 및 권한 체계

#### 관리자 역할 분류 (새로운 4단계 체계)
1. **최고관리자** (Super Admin)
   - 전체 시스템 관리
   - 관리자 계정 생성/삭제
   - 시스템 설정 변경

2. **관리자** (Admin)
   - 사용자 관리
   - 매장 관리
   - 컨텐츠 관리
   - 정산 관리

3. **매장관리자** (Business Manager)
   - 매장 승인/관리
   - 정산 조회

4. **컨텐츠관리자** (Content Manager)
   - 컨텐츠 관리
   - 이벤트 관리

#### 계정 생성 방식 ❌ 미구현
- **최고관리자**: 최초 수동 생성 (시스템 초기 설정) ❌
- **기타 관리자**: 최고관리자가 생성 ❌
- **임시 비밀번호**: 24시간 유효, 첫 로그인 시 변경 필수 ❌
- **2FA**: 모든 Admin 계정 필수 ❌ 미구현

**구현 필요사항:**
- 2FA 시스템 (TOTP)
- 역할 기반 권한 관리
- 백엔드 인증 API

### 핵심 가치 제안
- 리퍼럴 성과 실시간 모니터링
- 효율적인 매장 및 정산 관리
- 템플릿 기반 컨텐츠 관리

### 통합 컨텐츠 관리 시스템
모든 Buzz 앱 화면의 컨텐츠를 Admin에서 완전히 제어하는 통합 관리 시스템

```
📝 컨텐츠 통합 관리
├── 🏠 홈 화면 관리
│   ├── 추천 매장 설정 (순서, 노출 조건)
│   ├── 메인 배너/캐러셀 관리
│   ├── 카테고리 활성화/비활성화
│   └── 실시간 노출 제어
│
├── 🗺️ 지역추천 관리
│   ├── 컨텐츠 작성 (템플릿 기반)
│   ├── 노출 순서 설정
│   ├── 계절별 특집 설정
│   ├── 태그 관리
│   └── SNS 공유 설정
│
├── 🎪 이벤트 관리
│   ├── 이벤트 생성/수정/삭제
│   ├── 노출 기간 설정
│   ├── 참여 조건 설정
│   ├── 자동 종료 규칙
│   ├── 당첨자 관리
│   └── 통계 및 분석
│
├── 🎓 마케터 컨텐츠
│   ├── 교육 자료 업로드
│   ├── 리퍼럴 캠페인 설정
│   ├── 보상 정책 설정
│   └── 성과 랭킹 관리
│
├── 💬 커뮤니티 관리
│   ├── 게시판 카테고리 설정
│   ├── 공지사항 작성
│   ├── 부적절 컨텐츠 관리
│   └── 사용자 신고 처리
│
├── ⭐ 리뷰/평점 관리 ✅ 신규 추가
│   ├── 전체 리뷰 현황 대시보드
│   ├── 리뷰 승인/숨김/삭제 처리
│   ├── 신고된 리뷰 관리
│   ├── 매장별 평점 통계 관리
│   ├── 부적절한 리뷰 자동 감지
│   ├── 리뷰 검색 및 필터링
│   └── 대량 리뷰 처리 도구
│
├── 🎯 팝업 배너 관리
│   ├── 팝업 배너 생성/수정/삭제
│   ├── 노출 위치 설정 (center/bottom/top/fullscreen)
│   ├── 노출 빈도 설정 (항상/일1회/주1회/최초1회)
│   ├── 타겟팅 설정 (전체/조건부)
│   ├── 자동 닫힘 시간 설정
│   ├── 애니메이션 효과 설정
│   └── 노출 통계 분석
│
└── 🔔 실시간 관리
    ├── 긴급 공지 발송
    ├── 팝업 즉시 노출
    ├── 컨텐츠 즉시 숨김
    └── A/B 테스트 설정
```
